# CellD

[![version](https://img.shields.io/github/v/release/davidchangok/CellD)](https://github.com/davidchangok/CellD/releases)
[![MIT License](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/davidchangok/CellD/blob/master/LICENSE)

CellD is a fork of [Cell](https://github.com/enderneko/Cell) by enderneko, continuing development for WoW Retail (12.0.5+ Midnight). The original author is no longer actively maintaining Cell, so this fork keeps the addon updated and working for the latest WoW retail version.

CellD is a nice raid frame addon inspired by several great addons, such as **CompactRaid**, **Grid2**, **Aptechka** and **VuhDo**.
With a more human-friendly interface, CellD can provide a better user experience.
Hope you enjoy.

&nbsp;

## Features

- **Layouts:** auto switch layout by spec/role, supports party, raid, arena, and battleground.
- **Customizable Appearance:** textures, colors and alphas.
- **Built-in Click-Castings:** supports keyboard and multi-button mouse.
- **Indicators:** dozens of built-in indicators and unlimited custom indicators (icon, bar, rect, text, icons).
- **Raid Debuffs:** debuffs priority and glow.
- **Useful Raid Tools:** ready check, countdown, rebuff, death report, marks, battleres.
- **Nice Options UI:** I mean yes it's pretty darn good!
- **Spotlight Frame:** extra 15 unit buttons, can be set to Target, Focus, Unit, Tank, etc.
- **Quick Assist:** for Augmentation Evokers!
- **Compatibility:** [BigDebuffs](https://www.curseforge.com/wow/addons/bigdebuffs), [Class Colors](https://www.curseforge.com/wow/addons/classcolors), [OmniCD](https://www.curseforge.com/wow/addons/omnicd) and of course, [WAs](https://wago.io/weakauras).

&nbsp;

## Supported WoW Version

- **WoW Retail 12.0.5+ (Midnight)** only. Classic versions are not supported.

## Code Snippets

**Try disabling all snippets if there are any issues.**

Check <https://github.com/enderneko/Cell/tree/master/.snippets> for more snippets.

### Usage

1. CellD Options -> About -> Code Snippets
2. New -> Paste -> Save -> Tick the AutoRun box
3. Reload

&nbsp;

## Slash Commands

Use **/celld** for more information. (The old **/cell** command also works for backward compatibility.)
