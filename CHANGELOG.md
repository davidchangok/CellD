# CellD Changelog

## v1.0.0 (2026-06-13)

- Initial release of CellD, forked from Cell r277-beta by enderneko
- Renamed addon to CellD (Author: David W Zhang)
- Now supports only WoW Retail 12.0.5+ (Midnight); Classic version support removed
- Updated all path references from `Interface\AddOns\Cell\` to `Interface\AddOns\CellD\`
- Slash command: `/celld` (backward compatible `/cell` still works)
- Simplified version detection — removed all Classic/Wrath/TBC/Vanilla/Cata/Mists code branches
- Removed unused classic-specific files (~50 files)
- Secret Values / Opaque types handling preserved and enhanced for Midnight combat safety
- Updated documentation and links to point to https://github.com/davidchangok/CellD
